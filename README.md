Combinations
============

Smart algorithms about combinations and generators.

Expected input:

```
{
    'format': ['M', 'L'],
    'date': ['2020', '2021']
}
```

sample output:

```
[
   {
      'format': 'M',
      'date': '2020'
   },
   {
      'format': 'M',
      'date': '2021'
   },
   {
      'format': 'L',
      'date': '2020'
   },
   {
      'format': 'L',
      'date': '2021'
   }
}
```
